#### **Requisitos** 



1\. Gestão de Calendário: O sistema deve permitir criar jogos, definindo data, hora, estádio.



2\. Gestão de Arbitragem: O sistema deve permitir alocar uma equipa de arbitragem (árbitro principal, assistentes e VAR) a cada jogo, garantindo que não há conflitos de nacionalidade.



3\. Emissão de Bilhetes: Disponibilidade de bilhetes por jogo com a opção de emitir um bilhete.



4\. Alojamento: O sistema deve permitir associar cada seleção a um Centro de Treino/Hotel.



5\. Resultado Final: O sistema deve permitir registar o resultado final.



6\. Registo de Estatísticas de Jogo: O sistema deve permitir registar os eventos detalhados de cada partida (posse de bola, remates à baliza, cantos, faltas, cartões).



7\. Ficha de Pré-Jogo: O sistema deve permitir registar o onze inicial e capitão de cada seleção para cada jogo, equipa de arbitragem, estádio e horário.



8\. Classificação: O sistema deve atualizar as tabelas classificativas de cada grupo (A a H), consoante o resultado de cada jogo para cada seleção (Vitória, Empate, Derrota, Golos Marcados, Golos Sofridos e Pontos).



9\. Perfil das Seleções Participantes: O sistema deve permitir consultar a ficha detalhada de cada seleção participante (Grupo, Ranking, Participações na Competição, Convocados).



10\. Melhores Marcadores:  O sistema deve permitir consultar os melhores marcadores e assistentes da competição.



11\. Informação dos Estádios: O sistema deve permitir consultar informação sobre os estádios, incluindo a localização (cidade), o nome e a capacidade total de espectadores.



12\. Registo de Prémios Individuais (Man of the Match): O sistema deve permitir a votação ou registo do melhor jogador em campo para cada partida.



13\. Gestão de Melhores Terceiros Lugares: O sistema deve mostrar quais os melhores terceiros lugares que avançam para a fase a eliminar.



14\. Gestão de Eliminatórias: O sistema deve gerar os confrontos das fases seguintes (Desaseis avos de final até à Final) com base nos resultados das classificações.



15\. Gestão de Fusos Horários: O sistema deve permitir selecionar o fuso horário para cada jogo.
